## ğŸ“š Topics Covered

### 1. Design Patterns
Explore all three major categories of design patterns with "Bad vs. Good" implementations to understand the *why* behind each pattern.

| Category | Patterns |
| :--- | :--- |
| **Creational** | Singleton, Factory Method, Builder, Abstract Factory, Prototype |
| **Structural** | Adapter, Bridge, Composite, Decorator, Facade, Proxy |
| **Behavioral** | Observer, Strategy, Command, State, Chain of Responsibility, Iterator |

### 2. Refactoring Techniques
Learn how to improve your codebase without changing its external behavior.
- **Composing Methods**: Extract Method, Inline Method, Extract Variable.
- **Moving Features between Objects**: Move Method, Move Field, Extract Class.
- **Organizing Data**: Self Encapsulate Field, Replace Data Value with Object.

   ```bash

   flutter pub get
   ```

   ```bash
   flutter run
   ```
